Kausani Rainfall Comparison Charts

These charts compare rainfall in Kausani and Almora (the Uttaranchal district in which Kausani is located) at daily and monthly frequencies, respectively, for the period of 1987 to 2002. The daily rainfall data for Kausani is shared by Mr. David Hopkins, a resident of Kausani. The monthly aggregate rainfall data is sourced from India Meteorological Department (IMD).


Charts:

How to use: Click on thumbnails to see full-size maps. Hit *Escape* to exit the lightbox.

Legends: The charts show juxtaposed bars to compare the daily rainfall data for Kausani (shown in blue bars) with the monthly rainfall data for the district of Almora (shown in gray bars). The widths of the blue bars and the gray bars differ because the former shows rainfall measurement for a day (for Kausani) and the latter shows the daily average rainfall for the month (for Almora). In other words, the entire blue area in each chart gives the aggregate annual rainfall for the year concerned according to Mr. David Hopkins' measurement (for Kausani), while the entire gray area gives the aggregate annual rainfall for the year concerned according to IMD's measurement (for the District of Almora).

Documentation:

The original IMD rainfall data for 1901-2002 comes in three MS Excel spreadsheets, and each of them contains data for a set of states (for the entire period). The data is organised as follows (with the state and district codes given in a separate sheet of the spreadsheet file):

State ID,District ID,Year,January,February,March,April,May,June,July,August,September,October,November,December
1,1,1901,398.24,331.86,563.32,757.26,944.61,802.88,1020.44,432.67,717.18,223.14,44.45,101.16
1,1,1902,255.22,331.97,746.78,962.14,954.47,1116.03,910.09,688.77,542.22,434.55,523.79,159.64
1,1,1903,607.9,589.12,692.71,594.51,830.91,817.7,1094.51,983.38,732.75,165.03,146.73,210.24

To reorganise this data into a format that would allow easy comparison with Kausani rainfall data shared by Mr. David Hopkins, the IMD data file containing data for the district of Almora (in the state of Uttaranchal) is opened and the appropriate subset of the data is taken. To select the subset for the district of Almora, first the numeric value of the *District.ID* variable is replaced by "Almora" (for easy reference) for all relevant rows (that is rows with *State.ID* value of 5, indicating Uttaranchal, and *District.ID* value of "9", indicating Almora).

d <- read.csv ("imd_states_1-10.csv")

d$District.ID [d$State.ID == "5" & d$District.ID == "9"] <- "Almora"

a <- subset(d, d$District.ID == "Almora")

As shown above, the IMD data is organised with separate columns for each month of the year. To make the charts, the data format needs to be changed where all the months are given as different values of the same variable. That is, instead of each row showing monthly rainfall counts for all months for a single year, each rows should give the monthly rainfall in a district for an unique combination of year and month. This is done using the melt command from the Rehape2 package of R.

library(reshape2)
am <- melt(a, id = c('State.ID', 'District.ID', 'Year'))

This converts the data to the following format:

"State.ID","District.ID","Year","variable","value"
5,"Almora",1901,"January",326.21
5,"Almora",1902,"January",55.73
5,"Almora",1903,"January",226.95
5,"Almora",1904,"January",170.56

A new *Month* column is generated that gives the numeric values of the months concerned, which will be later used to generate the corresponding dates.

am$Month[am$variable == "January"] <- "01"
am$Month[am$variable == "February"] <- "02"
am$Month[am$variable == "March"] <- "03"
am$Month[am$variable == "April"] <- "04"
am$Month[am$variable == "May"] <- "05"
am$Month[am$variable == "June"] <- "06"
am$Month[am$variable == "July"] <- "07"
am$Month[am$variable == "August"] <- "08"
am$Month[am$variable == "September"] <- "09"
am$Month[am$variable == "October"] <- "10"
am$Month[am$variable == "November"] <- "11"
am$Month[am$variable == "December"] <- "12"

To generate the dates, first a *Day* column is created with universal value of "1", and then a *Date* column is generated by combining the values of the *Day*, *Month* and *Year* columns for each row.

am$Day <- "01"

am$Date <- paste(am$Day, am$Month, am$Year, sep = "-")

am$Date <- as.Date(am$Date, format = "%d-%m-%Y")

Only required columns are selected and renamed, and a relevant subset of the rows (for the period 1987 to 2002) is taken to create a new data file.

a2 <- data.frame(am$Date, am$Monthly.Rainfall.Almora, am$Year, am$Month)

colnames(a2)[1] <- "Date"
colnames(a2)[2] <- "Monthly.Rainfall.Almora"
colnames(a2)[3] <- "Year"
colnames(a2)[4] <- "Month"

a3 <- subset(a2, a2$Year > 1986)

write.csv(a3, file = "almora_rainfall_1987-2002.csv")

The daily rainfall data for Kausani shared by Mr. David Hopkins comes in XLS format, with data for different years given in different sheets of the same spreadsheet file. Each of these sheets are first converted into separate CSV files. While converting, initial correction of wrongly spelt (such as wrong spelling of the months) and entered data (such as use of the alphabet "o" insted of '0' or zero) are undertaken manually. To reorganise the data for 1987, the original CSV file is opened and a year column with uniform value of "1987" is created.

k <- read.csv("kausani_rainfall_1987_01.csv")

k$Year <- "1987"

Like the structure of the IMD data, Mr. Hopkins' daily rainfall files also come organised in separate columns for each month. The melt command from the Rehape2 package of R is used to reorganise the data in a format where all the months are shown as different values of the same column, as above.

library(reshape2)
k2 <- melt(k, id = c('Year', 'X'))

The Kausani rainfall data files contain various types of symbols for indicating zero or insiginificant rainfall in a day, such as "Trace", "-", ".", and "_". All these symbols are standardised to "0" to generate the charts.

k2$value[k2$value == "-"] <- 0
k2$value[k2$value == "Trace"] <- 0
k2$value[k2$value == ""] <- 0
k2$value[k2$value == "."] <- 0
k2$value[k2$value == "_"] <- 0

The *Date* column is generated following the same procedure as above -- starting with creating a *Month* column containing the numeric value of the month for the row concerned, and then combining the *X* (which gives the day concerned), *Month* and *Year* columns to generate the *Date* column, which is then converted into a standard date column as specified in R.

k2$Month[k2$variable == "Jan"] <- "01"
k2$Month[k2$variable == "Feb"] <- "02"
k2$Month[k2$variable == "Mar"] <- "03"
k2$Month[k2$variable == "Apr"] <- "04"
k2$Month[k2$variable == "May"] <- "05"
k2$Month[k2$variable == "June"] <- "06"
k2$Month[k2$variable == "July"] <- "07"
k2$Month[k2$variable == "Aug"] <- "08"
k2$Month[k2$variable == "Sept"] <- "09"
k2$Month[k2$variable == "Oct"] <- "10"
k2$Month[k2$variable == "Nov"] <- "11"
k2$Month[k2$variable == "Dec"] <- "12"

k2$Date <- paste(k2$X, k2$Month, k2$Year, sep = "-")

k2$Date <- as.Date(k2$Date, format = "%d-%m-%Y")

The columns are renamed and removed (as needed), and the reorganised data file is saved.

colnames(k2)[2] <- "Day"
colnames(k2)[4] <- "Daily.Rainfall.Kausani"
k2$variable <- NULL

write.csv(k2, file = "kausani_rainfall_1987_02.csv")

The reorganised daily rainfall data file for Kausani (such as "kausani_rainfall_1987_02.csv") is opened in Leafpad (or Notepad) and the rows with "NA" values in the *Date* column are removed. These are dates that do not exist (such as 29th February 1987) but were created due to the organisation of the original Kausani rainfall data file.

Now to generate the rainfall comparison charts, the IMD rainfall data file for Almora is opened and a subset is taken for the year "1987".

a <- read.csv("almora_rainfall_1987-2002.csv")

a2 <- subset(a, a$Year == "1987")

The data now gives the aggregate monthly rainfall counts for each month of the year 1987. A new *Daily.Rainfall.Almora* column is generated that gives the average daily rainfall for each month of the year 1987, by dividing the aggregate monthly rainfall count for each month by the number of days in that month. To be noted here is that the aggregate monthly rainfall count for the month of February is divided by one additional day for leap years.

a2$Daily.Rainfall.Almora[a2$Month == "1"] <- with(a2, Monthly.Rainfall.Almora[Month == "1"]/31)
a2$Daily.Rainfall.Almora[a2$Month == "2"] <- with(a2, Monthly.Rainfall.Almora[Month == "2"]/28)
a2$Daily.Rainfall.Almora[a2$Month == "3"] <- with(a2, Monthly.Rainfall.Almora[Month == "3"]/31)
a2$Daily.Rainfall.Almora[a2$Month == "4"] <- with(a2, Monthly.Rainfall.Almora[Month == "4"]/30)
a2$Daily.Rainfall.Almora[a2$Month == "5"] <- with(a2, Monthly.Rainfall.Almora[Month == "5"]/31)
a2$Daily.Rainfall.Almora[a2$Month == "6"] <- with(a2, Monthly.Rainfall.Almora[Month == "6"]/30)
a2$Daily.Rainfall.Almora[a2$Month == "7"] <- with(a2, Monthly.Rainfall.Almora[Month == "7"]/31)
a2$Daily.Rainfall.Almora[a2$Month == "8"] <- with(a2, Monthly.Rainfall.Almora[Month == "8"]/31)
a2$Daily.Rainfall.Almora[a2$Month == "9"] <- with(a2, Monthly.Rainfall.Almora[Month == "9"]/30)
a2$Daily.Rainfall.Almora[a2$Month == "10"] <- with(a2, Monthly.Rainfall.Almora[Month == "10"]/31)
a2$Daily.Rainfall.Almora[a2$Month == "11"] <- with(a2, Monthly.Rainfall.Almora[Month == "11"]/30)
a2$Daily.Rainfall.Almora[a2$Month == "12"] <- with(a2, Monthly.Rainfall.Almora[Month == "12"]/31)

The *Date* column is once again formatted to ensure that it follows the date column standard specified by R.

a2$Date <- as.Date(a2$Date, format = "%Y-%m-%d")

Then the reoganised daily rainfall data file for Kausani is opened, the *Date* column is reformmated according to date column specifications of R, and Almora and Kausani rainfall data files are combined using the merge command, where the *Date* variable ensures that corresponding rows (that is rows referring to the same date) from both data files are combined accordingly. The all.x = "TRUE" argument ensures that all values of *Date* in the Kausani rainfall data file is kept in the merged file.

k <- read.csv("kausani rainfall data/kausani_rainfall_1987_02.csv")

k$Date <- as.Date(k$Date, format = "%Y-%m-%d")

m <- merge(k, a2, by.x = "Date", all.x = "TRUE", by.y = "Date")

Columns that are not needed anymore are removed from the merged data, and some columns are renamed.

m$X.x <- NULL
m$X.y <- NULL
m$X.1 <- NULL
m$Year.y <- NULL
m$Month.y <- NULL

colnames(m)[2] <- "Year"
colnames(m)[5] <- "Month"

Now the merged data file contains daily rainfall counts for Kasuani for all the days of the year, but contains the averge daily rainfall in Almora for the month concerned, only for the first day of each month. Values for each month of the *Daily.Rainfall.Almora* column is replaced by the value taken by the first day of that month, that is by the daily average rainfall for that month, as calculated earlier.


m$Daily.Rainfall.Almora[m$Month == "1"] <- with(m, Daily.Rainfall.Almora[Month == "1" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "2"] <- with(m, Daily.Rainfall.Almora[Month == "2" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "3"] <- with(m, Daily.Rainfall.Almora[Month == "3" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "4"] <- with(m, Daily.Rainfall.Almora[Month == "4" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "5"] <- with(m, Daily.Rainfall.Almora[Month == "5" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "6"] <- with(m, Daily.Rainfall.Almora[Month == "6" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "7"] <- with(m, Daily.Rainfall.Almora[Month == "7" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "8"] <- with(m, Daily.Rainfall.Almora[Month == "8" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "9"] <- with(m, Daily.Rainfall.Almora[Month == "9" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "10"] <- with(m, Daily.Rainfall.Almora[Month == "10" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "11"] <- with(m, Daily.Rainfall.Almora[Month == "11" & Day == "1"])
m$Daily.Rainfall.Almora[m$Month == "12"] <- with(m, Daily.Rainfall.Almora[Month == "12" & Day == "1"])

Finally, the rainfall data for Almora and Kausani are plotted together using the ggplot2 library of R.

library(ggplot2)
library(scales)

ggplot(m, aes(x = Date)) + geom_bar(stat = "identity", aes(y = Daily.Rainfall.Almora), fill = "#636363", colour = "#636363") + geom_bar(stat = "identity", aes(y = Daily.Rainfall.Kausani), fill = "#6195ED", colour = "#6195ED") + scale_x_date(breaks = date_breaks("1 month"), labels = date_format("%b-%Y")) + scale_y_continuous(limits = c(0, 175), breaks = c(0, 10, 20, 30, 40, 50, 75, 100, 125, 150, 175)) + theme(panel.background = element_rect(fill = "white"), panel.grid.major = element_line(color = "#eeeeee"), axis.line = element_line(color = "#666666"), axis.title = element_text(size = "18", color = "#666666"), axis.ticks = element_line(color = "#666666"), axis.text = element_text(color = "#666666", size = "16")) + xlab("") + ylab("Rainfall")
